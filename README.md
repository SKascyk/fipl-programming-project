## Симметричный шифр на основе RC4 и зависимой от ключа перестановки (RC4 + Key-Dependent Permutation Cipher)

### Описание
Программа реализует симметричный алгоритм шифрования, который комбинирует байтовую перестановку текста, зависящую от ключа, и потоковое шифрование RC4.

### Требования
- Python 3.6 или выше

### Установка и запуск
1. Скачать файлы *cipher.py* и *main.py* или склонировать репозиторий.
2. Запустить *main.py*.
3. Следовать инструкциям консольного меню.

### Использование
#### Шифрование
1. Выберите пункт *1. Зашифровать текст*.
2. Введите открытый текст (одна строка).
3. Введите ключ (любая непустая строка).
4. Программа выведет hex-строку – зашифрованный текст.
#### Расшифрование
1. Выберите пункт *2. Расшифровать текст*.
2. Введите полученную hex-строку.
3. Введите тот же ключ, что использовался при шифровании.
4. Программа выведет исходный текст.
